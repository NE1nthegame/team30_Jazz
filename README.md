# jazz_team_project 
В процессе работы над проектом мы:
- Провели глубокую очистку: удалили дублирующие и почти константные признаки, устранили пары с корреляцией >0.95 во всех группах, а оставшийся дрифт признаков (FpDensityMorgan3) исключили в улучшенном датасете.
- Расширили признаковое пространство до 254 переменных за счёт внутригрупповых статистик, полиномов, взаимодействий, взвешенных сумм функциональных групп и KMeans‑кластеров.
- Отобрали топ‑200 наиболее важных признаков с помощью быстрого LightGBM, снизив размерность и шум.
- Для линейных моделей применили винзоризацию и StandardScaler, для деревьев оставили сырые значения.
- Гиперпараметры LightGBM, CatBoost и XGBoost оптимизировали через Optuna на 5‑фолдовой кросс‑валидации, а для Ridge подобрали alpha перебором по сетке.
- На основе out‑of‑fold предсказаний шести базовых моделей построили MLP‑ансамбль (3 сети) и добавили его прогнозы в пул.
- Все OOF предсказания откалибровали изотонической регрессией и объединили с 20 наиболее информативными сырыми признаками в мета‑признаки.
- Финальной метамоделью выступила Ridge, обученная независимо для каждого таргета; постобработка хвостов (сжатие верхних процентилей IC50 и CC50, ограничение SI) дала дополнительное снижение RMSE.
- Итоговый пайплайн показал RMSE ~272, подтвердив эффективность комбинации продвинутого feature engineering, стекинга с калибровкой и оптимизации хвостов.
